03. Sobre Mim

Conheça um pouco mais sobre minha jornada e paixão pela programação

Desenvolvedor Front-End Apaixonado

Olá! Meu nome é Pedro Enrick, sou um desenvolvedor front-end júnior apaixonado por criar experiências web incríveis e interfaces intuitivas. Minha jornada na programação começou com curiosidade e se transformou em uma paixão genuína por resolver problemas através do código.

Minha História

Tenho estudado HTML, CSS e JavaScript para criar projetos práticos que me ajudam a entender melhor como a web funciona. Cada projeto é uma oportunidade de crescimento e aprendizado contínuo, onde posso aplicar novas tecnologias e explorar diferentes abordagens para resolver problemas reais.

Embora meu foco principal não seja seguir carreira como programador tradicional, vejo a programação como uma ferramenta poderosa para expandir meus conhecimentos, desenvolver minha criatividade e aplicar soluções tecnológicas inovadoras em projetos pessoais e profissionais.

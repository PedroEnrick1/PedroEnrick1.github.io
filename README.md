## 🚀 Como Executar

### Pré-requisitos
- Navegador web moderno
- Servidor web local (opcional)

### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/PedroEnrick1/portfolio.git
   cd portfolio
   ```

2. **Abra o projeto**
   - Abra o arquivo `index.html` diretamente no navegador
   - Ou use um servidor local:
   ```bash
   # Com Python
   python -m http.server 8000
   
   # Com Node.js
   npx serve .
   
   # Com Live Server (VS Code)
   # Instale a extensão Live Server e clique com botão direito no index.html
   ```

3. **Acesse o portfólio**
   - Local: `http://localhost:8000`
   - Ou abra diretamente o arquivo `index.html`

## 🔧 Personalização

### Alterando Cores
Edite as variáveis CSS no arquivo `style.css`:

```css
:root {
  --primary-color: #sua-cor-primaria;
  --secondary-color: #sua-cor-secundaria;
  /* ... outras variáveis */
}
```

### Adicionando Projetos
1. Adicione a imagem do projeto na pasta `img/`
2. Crie um novo card na seção projetos do `index.html`
3. Siga a estrutura existente dos outros cards

### Modificando Informações Pessoais
Edite diretamente no arquivo `index.html`:
- Nome e título na seção hero
- Descrição na seção sobre
- Informações de contato
- Links das redes sociais
